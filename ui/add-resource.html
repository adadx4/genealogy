Build a new catalogue entry. The form generates a JSON block — paste it into the newSpecific array in scripts/transform-data.js, then run node scripts/transform-data.js to add it to the catalogue. Back to search.

For adding a sub-collection under an existing resource (e.g. a new FamilySearch collection), edit the newSubCollections array in the same script directly — that case isn't supported by this form yet.
